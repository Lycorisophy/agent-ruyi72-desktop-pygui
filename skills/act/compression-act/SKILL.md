# Compression Skill

文件压缩和解压技能。

## 功能特性

- **压缩文件**：创建 ZIP 压缩包
- **解压文件**：解压 ZIP、7z、RAR 等格式
- **添加文件**：向现有压缩包添加文件
- **解压到指定目录**：自定义解压路径
- **列出内容**：查看压缩包内的文件列表
- **密码支持**：支持密码压缩和解压

## 触发词

- "压缩文件"
- "解压文件"
- "compression"
- "zip"
- "解压缩"

## 使用示例

```
用户：将 Downloads 文件夹压缩
助手：[创建 Downloads.zip]

用户：解压这个 rar 文件
助手：[解压到当前目录]

用户：解压到指定文件夹
助手：[解压到指定路径]

用户：查看压缩包内容
助手：[列出压缩包内所有文件]
```

## 支持的格式

- **压缩**：ZIP
- **解压**：ZIP、7z、RAR（需要安装对应程序）

## 技术实现

使用 Python 标准库和第三方库：
- `zipfile` - ZIP 文件操作
- `shutil` - 文件操作
- `subprocess` - 调用外部解压程序

## 依赖要求

- Python 3.8+
- 7z.exe（可选，用于解压 7z/RAR）

## 安装说明

1. 将此文件夹复制到 如意72 skills 目录
2. 推荐安装 7-Zip 以支持更多格式
3. 重启 如意72 即可使用
