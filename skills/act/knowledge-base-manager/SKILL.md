# Knowledge Base Manager Skill

个人知识库管理技能，基于"个人知识库模板"架构。

## 功能特性

### 1. 文档搜索
- 按关键词全文搜索知识库
- 按标签筛选文档
- 按分类浏览文档
- 支持正则表达式搜索

### 2. 标签管理
- 添加/修改/删除文档标签
- 标签建议（基于标签使用规范）
- 标签组合建议

### 3. 分类浏览
- **00-知识库索引**: 管理文档
- **01-工作领域**: 项目、流程、方案
- **02-技术领域**: 开发、规范、技术
- **03-学习成长**: 技能、考试、笔记
- **04-参考资料**: 常用、技术、其他
- **05-个人事务**: 目标、计划、总结
- **06-归档资料**: 历史、过期、备份
- **99-临时文件**: 待处理、过渡

### 4. 知识提炼指导
- **REF**: 提炼-快速参考卡
- **KM**: 提炼-知识手册
- **SS**: 提炼-解决方案集

### 5. 文档规范检查
- 命名规范检查
- 元数据规范（Frontmatter）
- 版本管理
