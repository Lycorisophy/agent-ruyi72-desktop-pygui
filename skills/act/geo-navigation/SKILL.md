# Geo Navigation Skill

地理导航技能，查询两地之间的路线和交通信息。

## 功能特性

### 1. 驾车路线
- 最快路线/最短路线选择
- 预计驾车时间
- 预计过路费（高速费）
- 实时路况信息

### 2. 公共交通
- 地铁/公交换乘方案
- 高铁/火车班次
- 预计耗时
- 票价信息

### 3. 步行路线
- 步行距离
- 预计步行时间
- 推荐步行路线

### 4. 其他出行方式
- 打车费用估算
- 网约车（滴滴、高德等）
- 长途客运

## 触发词

- "怎么去"
- "路线"
- "多少时间"
- "打车多少钱"
- "开车多久"
- "坐高铁"
- "公交"
- "地铁"
- "步行"
- "从...到..."
- "去...怎么走"

## 使用示例

### 查询驾车路线
```
用户: 合肥高新区到上海怎么开车？
AI: 驾车路线：
- 路线：沪陕高速/沪蓉高速
- 距离：约400公里
- 耗时：约4-5小时
- 高速费：约180-200元
- 油价：约200元（按百公里8L油计算）
```

###查询公共交通
```
用户: 合肥到上海高铁多久？
AI: 高铁信息：
- 班次：每日多趟
- 耗时：约2.5-3小时
- 票价：二等座约180元，一等座约290元
- 出发站：合肥南站/合肥站
```

### 查询公交地铁
```
用户: 从合肥高新区到合肥南站怎么走？
AI: 公共交通方案：
- 地铁：2号线直达，约40分钟，票价4元
- 公交：约1小时，票价2元
- 打车：约30分钟，费用约30-40元
```

## 实现方式

### 网络搜索
使用搜索引擎查询路线信息：
- 百度地图、高德地图路线查询
- 携程、12306火车票查询
- 本地宝公交查询

### 信息整合
- 整合多个来源的路线信息
- 提供多种方案供选择
- 标注信息来源

## 注意事项

- 交通信息可能变化，以实际查询为准
- 费用为估算值，实际可能有所不同
- 给出建议时说明"约"或"预计"
- 推荐多种方案供用户选择
- 恶劣天气提醒出行注意
