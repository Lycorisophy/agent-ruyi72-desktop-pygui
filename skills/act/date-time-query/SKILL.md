# Date Time Query Skill

日期时间查询技能。

## 功能特性

### 1. 日期时间查询
- 当前日期和时间
- 星期几
- 季度（Q1/Q2/Q3/Q4）
- 距离周末/月底/年底天数

### 2. 农历节气
- 农历日期（年、月、日）
- 二十四节气
- 传统节日（春节、元宵、端午、中秋等）
- 生肖、星座

### 3. 节假日查询
- 中国法定节假日
- 调休上班日
- 自定义节假日提醒

### 4. 天气查询
- 指定城市/地区当前天气
- 指定日期天气（未来7天）
- 温度、湿度、风力、空气质量
- 天气预警信息

## 触发词

- "今天几号"
- "现在几点"
- "查下天气"
- "今天农历"
- "什么节气"
- "今天星期几"
- "这季度"
- "距离周末"
- "过年"
- "五一"
- "春节"

## 使用示例

### 查询当前时间
```
用户: 今天几号？
AI: 今天是2026年3月13日，星期五，距离周末还有1天！
```

### 查询天气
```
用户: 北京今天天气怎么样？
AI: 北京今天：晴，15°C~25°C，东南风2级，空气质量良~
```

###查询农历节气
```
用户: 今天农历是什么？
AI: 农历：二月十四今日惊蛰，适宜春耕备耕~
```

## 实现方式

### 日期时间（使用系统工具）
使用Python datetime模块获取当前系统时间：
- datetime.now() 获取当前日期时间
- isocalendar() 获取周数和季度
- strftime() 格式化输出

### 农历节气（使用Python库或搜索）
- 使用Python库：lunardate、sxtwl等
- 或使用搜索引擎查询当日节气

### 天气查询（优先搜索）
- 使用网络搜索查询天气（免费）
- 如有API可使用和风天气、OpenWeatherMap等

## 注意事项

- 日期时间使用系统时间，无需网络
- 天气优先使用搜索查询免费信息
- 查询失败时尝试搜索或告知用户
