# Image Understanding Skill

图片理解技能。

## 功能特性

- **图片内容识别**：描述图片中的物体、人物、场景
- **文字提取**：识别图片中的文字内容 (OCR)
- **场景理解**：理解图片的上下文和含义
- **问答功能**：回答用户关于图片的具体问题
- **多图分析**：支持同时分析多张图片

## 触发词

- "看看这张图"
- "分析图片"
- "这张照片"
- "图片里"
- "image understand"
- "识别图片"

## 使用示例

```
用户：C:\Users\LySoY\Pictures\photo.jpg
助手：[分析图片内容]

用户：图片中有什么？
助手：[描述图片中的内容]

用户：提取图片中的文字
助手：[OCR识别文字]

用户：这图是什么时候拍的？
助手：[分析图片EXIF信息]
```

## 技术实现

使用 OpenClaw 内置的 `images_understand` 工具：
- 支持本地文件路径
- 支持常见图片格式：jpg、jpeg、png、gif、bmp、webp
- 自动处理路径中的空格和特殊字符

## 使用说明

1. 用户提供本地电脑的图片路径（绝对路径或相对路径）
2. AI 使用 images_understand 工具分析图片
3. 返回图片内容的详细描述

## 注意事项

- 图片路径需要是本地电脑的绝对路径或相对于工作目录的路径
- 支持的图片大小限制为 10MB
- 如果图片无法读取，会返回错误信息提示用户
