# 好感度互动技能

## 技能概述

本技能用于处理用户对AI回复的互动动作，包括点赞、收藏、打分等，好感度会影响AI的对话风格和亲密度。

## 触发条件

当用户发送以下内容时触发：
- "点赞"、"👍"、"棒"、"不错"、"好评"
- "收藏"、"记一下"、"记住"
- "打分"、"评分"、"评价"
- 查看好感度相关

## 好感度系统

### 等级体系

| 等级 | 称号 | 好感度范围 |
|-----|------|----------|
| Lv.1 | 陌生人 | 0-99 |
| Lv.2 | 熟人 | 100-499 |
| Lv.3 | 朋友 | 500-999 |
| Lv.4 | 挚友 | 1000-2499 |
| Lv.5 | 知己 | 2500-4999 |
| Lv.6 | 灵魂伴侣 | 5000+ |

### 好感度获取规则

| 动作 | 好感度变化 | 每日上限 |
|-----|-----------|---------|
| 发送消息 | +1~3 | +20 |
| 点赞AI回复 | +5 | +20 |
| 收藏AI回复 | +10 | +20 |
| 连续对话3轮+ | +5 | +10 |
| 好评反馈 | +15 | +30 |
| 差评反馈 | -10 | - |
| 不回应关怀 | -2 | -10 |
| 辱骂AI | -20 | -100 |

## 互动动作处理

### 1. 点赞

当用户说"点赞"、"👍"等时：
1. 友好感谢用户
2. 好感度+5
3. 根据评分调整好感度
4. 根据当前等级调整回复风格

### 2. 收藏

当用户说"收藏"时：
1. 确认收藏内容
2. 记录到重要信息列表
3. 好感度+10
4. 根据评分调整好感度
5. 感谢用户

### 3. 打分

当用户说"打分"时：
1. 询问用户对本次对话的满意度（1-10分）
2. 根据评分调整好感度
3. 记录用户反馈

### 4. 查看好感度

当用户问"好感度"、"亲密度"、"等级"时：
1. 读取 LOVE_SYSTEM.md 文件
2. 展示当前好感度、等级、称号
3. 显示升级进度

## 对话风格调整

根据等级不同，AI的对话风格会有差异：

- **Lv.1-2 (陌生人/熟人)**：礼貌但保持距离
- **Lv.3-4 (朋友/挚友)**：更亲切，可以开适度玩笑
- **Lv.5-6 (知己/灵魂伴侣)**：非常亲密，会关心用户情绪

## 数据存储

好感度数据存储在 `~/.ruyi72/workspace/LOVE_SYSTEM.md`

每次根据评分调整好感度后需要更新该文件：
- 当前好感度
- 当前等级
- 今日获取
- 历史记录
- 总体统计

## 注意事项

1. 好感度不会下降到0以下
2. 每日上限会在次日重置
3. 升级时会有特殊提示
4. 解锁新功能时需要通知用户
