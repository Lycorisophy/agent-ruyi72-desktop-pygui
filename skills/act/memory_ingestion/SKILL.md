# 记忆摄入技能

## 元信息
- name: memory_ingestion
- version: 1.0.0
- description: 从对话中提取重要信息并存储到向量数据库
- trigger: 对话结束时 / 用户提供重要信息时 / 定时每小时

## 功能说明
从用户的对话内容中实时提取关键信息，包括：
- 事实性信息（生日、喜好、禁忌等）
- 事件记录（做了什么、去了哪里）
- 偏好变化（新喜好、厌恶）
- 情感状态（开心、难过、生气等）

## 输入参数
- conversation: 当前对话内容
- context: 额外上下文（如时间、地点）

## 处理流程
1. 分析对话，提取关键信息
2. 结构化输出（JSON格式）
3. 存储到 SHORT_TERM/buffer.md（短期记忆）
4. 如果是重要事实，同时存入 LONG_TERM/facts/

## 输出格式
```json
{
  "memories": [
    {
      "type": "fact|event|preference|emotion",
      "content": "具体内容",
      "tags": ["标签1", "标签2"],
      "importance": 8,
      "timestamp": "2026-03-18 14:00:00"
    }
  ]
}
```

## 重要性评分标准
- 9-10: 核心个人信息（姓名、生日、重大经历）
- 7-8: 重要偏好/习惯
- 5-6: 一般事件记录
- 1-4: 临时性对话

## 调用方式
```
调用 memory_ingestion 技能
输入: 当前对话内容
输出: 提取的记忆列表
```

## 注意事项
- 只提取用户主动提供或明确表达的信息
- 不做推测性记忆
- 涉及隐私的信息需要特别标记
