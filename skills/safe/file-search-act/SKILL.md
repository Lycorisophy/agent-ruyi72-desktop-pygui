# File Search Skill

文件全文搜索技能。

## 功能特性

- **全文搜索**：在文件中搜索指定关键词
- **多格式支持**：支持 txt、md、py、js、html、css、json、xml 等
- **正则搜索**：支持正则表达式搜索
- **结果预览**：显示匹配行的上下文
- **文件类型过滤**：按扩展名过滤搜索范围
- **大小限制**：可设置搜索文件的大小上限

## 触发词

- "搜索文件"
- "查找内容"
- "file search"
- "全文搜索"
- "grep"

## 使用示例

```
用户：在 Downloads 文件夹搜索 "如意72"
助手：[搜索并显示所有匹配行]

用户：查找包含 TODO 的文件
助手：[列出所有 TODO 标记]

用户：在项目目录搜索函数定义
助手：[使用正则表达式搜索函数名]

用户：只搜索 Python 文件
助手：[指定扩展名为 .py]
```

## 技术实现

使用 Python 标准库：
- `os` - 遍历文件系统
- `re` - 正则表达式匹配
- `pathlib` - 路径操作

## 依赖要求

- Python 3.8+

## 安装说明

1. 将此文件夹复制到 如意72 skills 目录
2. 无需额外依赖
3. 重启 如意72 即可使用
