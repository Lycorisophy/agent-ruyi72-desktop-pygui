# Disk Manager Skill

Windows 磁盘管理技能，提供磁盘和分区的查看与管理功能。

## 功能特性

- **查看磁盘**：列出所有物理磁盘及其基本信息
- **分区详情**：显示磁盘分区结构和大写
- **空间使用**：显示各分区的已用空间和可用空间
- **驱动器信息**：获取驱动器类型、文件系统、卷标
- **磁盘健康**：检测磁盘状态和 SMART 信息
- **目录大小**：计算指定目录的占用空间

## 技术实现

使用 PowerShell 命令：
- `Get-Disk` - 获取物理磁盘信息
- `Get-Partition` - 获取分区信息
- `Get-Volume` - 获取卷信息
- `Get-PSDrive` - 获取驱动器信息
- `Get-ChildItem` - 计算目录大小

## 依赖要求

- Python 3.8+
- Windows PowerShell 5.1+
- Windows 10/11 或 Windows Server 2016+
