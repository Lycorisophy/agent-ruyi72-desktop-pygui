本目录供「知识库管理」会话在运行时读取，拼入系统提示，**无需**用户复制到本仓库。

## 与外部模板的关系

若你本机已有「个人知识库」类文件夹，可将**工作区**直接设为该目录或其副本；内置文案中的目录名（收件箱、加工、归档等）仅为常见习惯，可与你的模板对齐或忽略。
